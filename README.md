# 🏧 ATM System (C++ Project)

A simple ATM simulation system built using C++ and Object-Oriented Programming (OOP).

---

## 💡 Features
- User login with PIN authentication (3 attempts)
- Check account balance
- Deposit money
- Withdraw money with validation
- Menu-driven system using loops and switch case

---

## 🧠 Concepts Used
- Object-Oriented Programming (Classes & Objects)
- Encapsulation
- Functions
- Conditional statements (if/else)
- Loops (while, do-while)
- Switch case

---

## 🚀 How It Works
1. User enters PIN
2. If PIN is correct → access granted
3. User can choose:
   - Check Balance
   - Deposit Money
   - Withdraw Money
   - Exit system
4. Balance updates dynamically after each operation
